# 设计思想

万物皆模块
A bundler for javascript and friends

## 功能
1. 打包散落文件
2. 兼容转译语法
3. 打包周边静态文件

## 思路
我们可以把应用初次加载所必需的模块打包到一起，其他的模块再单独打包，等到应用工作过程中实际需要用到某个模块，再异步加载该模块，实现增量加载，或者叫作渐进式加载，非常适合现代化的大型 Web 应用。
